# wsc-image — AI 视觉创意专家

> 把你脑子里模糊的画面，变成 AI 能精准执行的提示词。

不管是「画一张赛博朋克风海报」还是「帮我把这张照片改成电影感」，直接说需求就行——不用背命令，不用懂提示词语法。

---

## 作用与定位

大多数人用 AI 画图，卡在「不知道怎么描述」这一步。说太少，出来的图不对；说太多，AI 理解不了。wsc-image 解决的就是这个问题：

1. **生图提示词** — 你说一句话，它给你一段经过 7 层结构优化的专业提示词，能直接用
2. **视频分镜** — 描述一个场景或故事，输出完整分镜表和每个镜头的运镜提示词
3. **参考图分析** — 上传一张你喜欢的图，它从风格、色调、光影、构图四个维度拆解，再问你想保留哪些
4. **修图 / P图** — 提供原图，说「换背景」「保留人物」「电影感重建」，自动生成保护五官比例的修改指令

---

## 四大场景，怎么判断走哪条路

AI 会根据你说的话自动判断场景，不需要你手动指定：

```
你发了图片？
  ├─ 说「修图 / 换背景 / 保留人物 / 调色 / 电影感」
  │    → 场景 D：修图 / P图提示词
  │       ├─ 修图模式：增强光影氛围，不换主体
  │       └─ P图模式：换背景场景，锁定五官不变
  │
  └─ 说「参考这个风格 / 类似这个 / 照这个来」
       → 场景 C：参考图分析
            四维拆解后弹窗确认：
            A. 保留整体风格 → 只换主体
            B. 只要光影质感 → 融合新主题
            C. 只是找灵感 → 重新按文字生成
            D. 自定义保留元素

你在描述内容？
  ├─ 静态画面（人物 / 产品 / 场景 / 概念图）
  │    → 场景 A：AI 绘图提示词
  │       7层结构：主体→细节→光影→构图→环境→风格→参数
  │
  └─ 动态内容（运镜 / 视频 / 故事 / 动画）
       → 场景 B：视频分镜提示词
            输出分镜表 + 每个镜头的完整描述
```

**信息不够时**，会自动弹出选择题引导你（风格 / 比例 / 节奏），选一下就继续，不用你自己想。

**结果不满意时**，说「差一点」「再优化」「不够电影感」，会追问具体哪里不对，精准调整那一层，其他不动。**没有轮数限制。**

## 怎么使用

### 安装（Claude Code）

把 `wsc-image` 文件夹放进 skills 目录：

```
~/.claude/skills/wsc-image/
```

### 触发

直接说需求，不用记命令：

| 你说的话 | 进入场景 |
|---------|---------|
| 「帮我画一张赛博朋克风的城市夜景」 | 场景 A — 弹窗问比例和风格细节 |
| 「做一个产品发布的视频提示词」 | 场景 B — 生成分镜表 |
| 「参考这张图，画一个类似氛围的人像」 | 场景 C — 四维分析 + 意图确认 |
| 「帮我把这张照片修成电影感，不要改人物」 | 场景 D — 修图模式 |
| 「把这张图的背景换成洱海，保留人物」 | 场景 D — P图模式，弹窗确认保留范围 |

### 迭代

```
你：差点意思，光影不够强
AI：【需要调整哪里？】
    A. 主体描述  B. 色彩/光影  C. 风格方向  D. 构图  E. 氛围  F. 其他
你：B
AI：[输出增强光影版本]
```

## 把收集的内容喂给 AI 并保存

如果你收集了一批提示词、文案或技巧文档，想让 AI 帮你整理成可以随时调用的知识，流程如下：

### Step 1：准备素材

把收集到的内容整理好，格式不限：
- 复制的文章 / 文案段落
- Markdown 文件（比如 `修图P图.md`）
- 截图里的文字（复制出来）
- 任意平台的提示词样例

### Step 2：发给 AI，说清楚意图

```
以下是我收集的关于 [主题] 的内容，请帮我：
1. 提取核心规则、使用逻辑和关键提示词结构
2. 整理成结构化的 Markdown 笔记
3. 保存到 memory/image_patterns.md（或我指定的路径）
```

### Step 3：AI 蒸馏后写入 memory

AI 会把散乱内容压缩为：
- **规则层**：触发条件 → 执行逻辑 → 输出格式
- **词汇层**：高频关键词 / 专业术语提炼
- **样例层**：代表性提示词片段

写入 `memory/` 目录后，下次对话时 AI 会自动加载，不用重复喂。

### 用这套流程扩展 wsc-image

发现新的提示词技巧或平台更新？同样的方式发给 AI，让它蒸馏后追加进 `memory/image_patterns.md`，skill 就跟着进化。

## 平台支持

| 平台 | 语言 | 默认 |
|------|------|------|
| 即梦 4.0 / 4.5 | 中文自然语言 | ✅ 默认输出 |
| 豆包 | 中文自然语言 | 指定后输出 |
| Midjourney | 英文 + `--ar --v --q` 参数 | 指定后输出 |
| Stable Diffusion SDXL | 英文，双编码器优化 | 指定后输出 |
| Stable Diffusion 1.5 | 英文 Tag 堆叠 + 负向提示词 | 指定后输出 |
| 可灵 / 即梦视频 / Sora | 中/英，镜头语言 | 指定后输出 |

```
├── templates/
│   ├── image.md          # 图片提示词模板
│   └── video.md          # 视频分镜模板
└── memory/

    ├── video_patterns.md # 视频运镜高频模式
    └── iteration_log.md  # 迭代优化记录（自动追加）
```
