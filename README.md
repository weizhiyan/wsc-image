- **[wsc-skill-ux](https://github.com/weizhiyan/wsc-idea)** — UI/UX 设计与需求专家（界面 · 交互优化）

## 能做什么

| 场景 | 触发方式 | 输出 |
|------|---------|------|
| **场景 A** 生图提示词 | 描述静态画面 | 7层结构提示词，适配各平台 |
| **场景 B** 视频提示词 | 描述动态内容/运镜 | 分镜表 + 完整视频提示词 |
| **场景 C** 参考图分析 | 上传图片 + 「参考这个风格」 | 四维风格分析 + A/B/C/D 意图确认 |
| **场景 D** 修图/P图 | 上传图片 + 「修图/换背景/保留人物」 | 修图增强或P图合成提示词，五维保护原则 |

---

## 核心机制

**弹窗引导** — 信息不足时自动弹出选择题，不打断对话节奏

**无限迭代** — 说「差点意思」「再优化」，持续追问直到满意，无轮数限制

**平台适配** — 默认输出即梦中文版；指定平台时自动切换格式（MJ 英文参数 / SDXL 双编码器 / SD1.5 标签堆叠）

**五维保护（修图/P图）**
```
① 保留优先  — 严格保留五官、脸部比例、肤色
② 原比例    — 不改变画面宽高比
③ 无缝融合  — 新元素与主体光影/色调/透视匹配
④ 不过度处理 — 保留真实皮肤纹理，不网红化
⑤ 拒绝漂移  — 不改变年龄、种族、骨相特征
```

## 平台支持

| 平台 | 语言 | 提示词风格 |
|------|------|-----------|
| 即梦 / 豆包 | 中文 | 自然语言，描述性（默认） |
| Midjourney | 英文 | 关键词堆叠 + `--ar --v --q` 参数 |
| Stable Diffusion SDXL | 英文 | 自然语言短句 + 双编码器优化 |
| Stable Diffusion 1.5 | 英文 | Tag 式堆叠 + 负向提示词 |
| DALL-E | 中/英 | 完整描述句 |
| 可灵 / 即梦视频 / Sora | 中/英 | 镜头语言 + 动态描述 |

## 安装

将 `wsc-image` 文件夹放入 Claude Code 的 skills 目录：

```
~/.claude/skills/wsc-image/
```

直接描述需求即可触发，无需记忆命令。
