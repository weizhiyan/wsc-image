# 海报、UI 图标与 3D 主视觉模式库

本文档总结了生成高质量、商业级平面海报、UI 界面图标以及复杂 3D 数据可视化/产品主图的提示词模式。

## 一、高级电影/商业海报模式 (Cinematic Poster)

**核心要素**：
`极端的光影切片` + `强烈的色彩碰撞` + `极简的负空间构图` + `高级摄影质感`

### 优质样例：
1. **红蓝撞色·舞台剪影风（汽车/高定商业）**
   > 高定商业广告大片摄影。一辆流线型质感的墨绿色运动SUV停在画面左侧，右侧站着一位身穿长裙的女性模特。极简的舞台化空间设计，背景由巨大的几何墙面构成。极具戏剧性的高饱和对比光影：画面左半部分被神秘的深青色/孔雀蓝冷光笼罩，右侧背景墙被一束强烈的朱红色/橘红色暖光切面照亮，形成极端的红蓝冷暖撞色。人物站在强烈的红蓝交界处，完全背光，呈现出高级、纯粹的黑色剪影效果（Silhouette）。汽车表面的金属车漆反射出周围的红蓝冷暖光泽。低视角，相机贴近地面，画面大量留白（负空间），充满高级感与科幻感，电影级调色，大师级用光，8K分辨率。

2. **微观极度拉近·竞技动态模糊（体育/极限动作）**
   > 前景无限拉近，将运动员的冰球杆(杆身带有碳纤维纹理、握柄缠布与击球部位磨损细节)与高速滑行中的冰球(橡胶球体表面划痕清晰、因急停溅起的冰屑飞散)放大至占据画面70%视觉空间，作为绝对视觉中心，突出球杆挥击时的动态模糊与冰球向镜头急速冲来的轨迹(鱼眼畸变使球杆边缘呈自然弧形)，强化近景压迫感；中景动态定格，捕捉主角侧身发力击球的瞬间；远景极致压缩，将远处布满呐喊观众的冰雪看台与球场护栏缩至画面上方窄区域，鱼眼畸变让背景边缘弧形弯曲，通过大小对比凸显前景球杆与冰球的巨大视觉冲击力。

3. **微缩景观·笔墨浮雕融合（城市文旅/艺术大作）**
   > 一支正在绘画的毛笔，描摹出一条由左下到右上的细笔迹，曲线，笔迹里是微观的哈尔滨的各大景点景观，油画风格，学院派的版式设计海报，有着大师级排版的文字设计，3D，微景观，厚涂，厚薄不同塑造立体感，暖色橙红上是哈尔滨相关元素(圣索菲亚大教堂、雪花与欧式建筑轮廓，远处夕阳余晖)，极简俯视图，大面积留白，白色纹理纸质背景。主标题:“HARBIN”艺术字体，非常醒目。古色古香的欧式建筑与冰雪元素点缀其间，画面呈现出立体浮雕效果，毛笔笔尖在笔迹的尽头，笔迹之外都是留白，整体风格融合了传统中国风与现代立体艺术，高细节，超写实，HDR，8K分辨率。

## 二、UI 界面与系统图标设计 (UI Icons)

**核心要素**：
`等距/斜45度透视` + `特定材质（如磨砂玻璃、塑料、金属）` + `渐变光影与色彩` + `背景极度纯净`

### 优质样例：
1. **柔和极简风（浅色系塑料/马卡龙色）**
   > 画一个极简主义3D【打字机】图标，柔和的阴影，光滑的塑料纹理，柔和的粉彩色调，如米色、灰色和橄榄绿。柔和的灯光。等距视图，具有微妙的透视，可爱而友好的风格，现代数字渲染。纯白背景。

2. **通透渐变风（办公数字化系统/蓝绿低保和）**
   > 清新科技的图标风格，扁平化，多细节，高质量，采用蓝白配色(带一点绿)低饱和度的色调，元素具有通透的渐变层次与立体质感，科技数字化管理场景。图标造型简约科技感且紧扣工业场景核心元素，表意精准。多彩低饱和度色调协调统一，多层次强化场景的专业感，适合用于工业类 APP 的功能分类或界面导航场景。纯白背景，3D 轻立体斜角透视视角，表面带细腻磨砂光泽，边缘搭配柔和高光勾勒轮廓，融入同色系深浅渐变层次，强化轻量立体与高级质感。

3. **复古未来主义（Flux / 复杂复古电脑 UI）**
   > flux提示词：3d subject, pokemon, Super cool vision, Retro-futuristic pastel computer, soft cream and white gradient background, vibrant orange/green accents, CRT-style circular screen with pixelated emoji and thick curved bezel, chunky mechanical keyboard with blocky keys, floating discount stickers, playful smiley face decal, smooth plastic textures, gentle ambient lighting, soft reflections, Cinema 4D render, nostalgic 80s vibe, high-detail 3D design, warm and inviting aesthetic, interactive UI elements, isometric front view, studio lighting, soft shadows, pastelcore aesthetic

## 三、3D 科技主图与数据可视化 (Tech & Data Visualization)

**核心要素**：
`全景/等距视图构图（底座、堆叠、悬浮）` + `科技材质（发光流、冷金属、水晶、能量纹理）` + `深色渐变宇宙/数字空间背景` + `立体文字与光影交互`

### 优质样例（极度详细的主图构建）：

1. **悬浮数据底座与螺旋DNA结构（智慧数据分析）**
   > 右侧主图采用底座型构图，呈现悬浮在蓝紫色能量底座上的螺旋状数据DNA结构，结构由发光的数据流、动态旋转的柱状图、闪烁的二进制代码交织而成，底座表面布满流动的电路纹理。主图细节表现为数据流的动态拖影、图表边缘的金属光泽、代码矩阵的像素化闪烁，材质质感为半透明能量光膜与冷金属混合，整体空间结构为“前景数据DNA突出，中景底座支撑，背景为渐变的深邃宇宙”。背景采用深蓝色到紫色的渐变，叠加星云与数据流纹理，增强科技氛围与视觉纵深。

2. **立体堆叠数据塔与能量矩阵（数据大屏/服务宣发）**
   > 右侧主图采用堆叠型构图，展示由多个不规则数据立方体堆叠而成的数据塔，每个立方体表面布满动态变化的条形图、散点图、数据表格等信息，立方体之间由金色的数据流连接，底座为黑色的能量矩阵，带有红色的能量波纹动理。主图细节表现为立方体表面信息的实时刷新、数据流的流动特效、立方体边缘的发光轮廓，材质质感为金属磨砂与半透明能量胶质混合，整体空间结构为“前景数据立方体细节丰富，中景堆叠的数据塔主体，背景为模糊的未来城市夜景”。背景采用深紫色到黑色的渐变，叠加霓虹灯光与城市剪影。

3. **立体系统防护盾与防火墙代码（网络安全宣传）**
   > 右侧主图为悬浮在带有蓝色能量纹路底座上的立体系统防护盾，盾牌表面布满动态的防火墙代码、闪烁的加密锁图标、流动的数据流波纹等元素，采用底座型构图，底座呈现半透明的水晶质感。主图细节表现为防护盾表面的代码微微发光、加密锁图标边缘带有金属光泽、数据流波纹具有流动的光影效果，材质质感为冷金属与透明能量光膜混合。背景采用深蓝色到黑色的渐变，叠加二进制代码雨纹理，增强科技氛围。光影设计采用顶部冷光主照明 + 底座反光，并结合防护盾表面的光膜反光效果，突出主图的立体感和质感。装饰性光效为围绕防护盾的蓝色光带和底座边缘的粒子光晕。
