# Logo 设计提示词模式库

本文档总结了生成高质量、极简、高级感 Logo 的核心提示词结构和优质样例。

## 核心规则与结构

优秀的 Logo 设计提示词通常遵循以下精简、并列的短句结构（尤其适用于即梦/豆包等国内大模型）：

`[基本定调] + [主体意象及融合方式] + [英文字体/内容] + [风格/工艺/质感] + [排版/背景]`

**1. 基本定调（开篇锚定方向）**
- `logo设计` / `logo品牌标志`
- `极简` / `简约LOGO设计`
- `抽象风格的科技公司标志`

**2. 主体意象及融合方式（核心创意）**
- 明确两个或以上元素的融合：`[元素A]和[元素B]的完美/创造性融合`
- 例如：`猫与咖啡的融合`、`小鳄鱼和皮包完美结合`、`松树和山石元素`、`兔子头部造型与寿司的三角形创造性的融合`

**3. 字体与文字内容**
- `写着英文“[文字内容]”` / `用英文公司名设计成logo`
- 字体修饰：`流线型字体` / `手绘简笔画风格` / `漫画字体`

**4. 风格、工艺与质感词汇（拉升格调的关键）**
- **高级质感类**：`丝印冷烫`（极强的高级感触发词）、`国际化`、`高级感`
- **形态类**：`扁平化`、`极简`、`负空间设计`、`抽象`、`大气`
- **特殊风格类**：`手绘简笔画风格`、`涂鸦`、`儿童插画`、`参考毕加索风格`、`现代艺术设计风格`
- **动态/光影类**：`流星滑过动态效果`、`透明感和叠加效果`、`光线均匀`

**5. 排版、背景与全局控制**
- `纯白色背景` / `黑色背景`
- `排版专业` / `对比合理` / `细节清晰` / `设计大师作品` / `设计感`

## 优质样例库 (Copy & Paste)

### 1. 国际高级·丝印冷烫风（适用：品牌升级/轻奢/国际化企业）
> **公式**：logo品牌标志，极简，图型设计，写着英文“[名字]”，[元素A]与[元素B]的融合，国际化，丝印冷烫，高级感，扁平化，设计感。
* **样例 (饮品)**：logo品牌标志，极简，图型设计，写着英文“DREAMINA"，猫与咖啡的的融合，国际化，丝印冷烫，高级感，扁平化，设计感
* **样例 (风景/文旅)**：logo品牌标志，极简，图型设计，写着英文“TAISHAN",图形是融合了松树和山石元素，国际化，丝印冷烫，高级感，扁平化，设计感。

### 2. 科技动态·流线抽象风（适用：SaaS/科技/AI/通讯）
> **公式**：logo设计，一个抽象风格的科技公司标志logo设计，[主体]抽象设计，采用流线型字体，搭配[动态修饰，如流星滑过动态效果/透明叠加效果]，扁平化极简风格。画面光线均匀，细节清晰，强调了标志的辨识度和专业性，设计大师作品，排版专业，对比合理。
* **样例 (抽象字母)**：logo设计，一个抽象风格的科技公司标志logo设计，字母A抽象设计，采用流线型字体，搭配流星滑过动态效果，扁平化极简风格。画面光线均匀，细节清晰，强调了标志的辨识度和专业性，设计大师作品，排版专业，对比合理。
* **样例 (多色叠加)**：一个简洁现代的科技公司标志，名为“renovo”，下方辅以小字“TECHNOLOGIES”。标志上方是由红、绿、蓝三色交织而成的抽象图案，形似火焰或花瓣，具有透明感和叠加效果，整体呈现出活力和创新感，背景为纯白色。

### 3. 趣味融合·简笔插画风（适用：宠物/母婴/创意品牌/餐饮）
> **公式**：简约LOGO设计，用[手绘简笔画/涂鸦/儿童插画]风格，[动物/人物]和[行业物品]完美结合的作品，英文字“[名字]”，体现[某某行业]理念，抽象，大气，简洁，扁平化，纯白色背景。
* **样例 (皮具)**：简约LOGO设计，用手绘简笔画风格，可爱小鳄鱼和皮包完美结合的作品，英文字“pibao”，体现皮包行业理念，抽象，大气，简洁，扁平化，纯白色背景
* **样例 (宠物)**：LOGO设计，用抽象胖胖猫和文字“Chubby Cat”完美结合的作品，体现宠物行业理念，抽象艺术，用现代艺术设计风格体现极具宠物元素的图案，扁平化，纯白色背景，极简
* **样例 (餐饮)**：LOGO设计，品牌标志，极简，图型设计，兔子头部造型与寿司的三角形创造性的融合，简约协调，涂鸦，儿童插画，参考毕加索风格，扁平化，设计感，白色背景。

### 4. 漫画字体·二次元风（适用：娱乐/潮玩/社交/自媒体）
> **公式**：字体，logo设计，漫画字体，字体内容“[文字]”，漫画纹理，[情感修饰，如胖嘟嘟可爱感觉/热血张力]，背景控制，卡通感十足。
* **样例 (可爱元气)**：字体，logo设计，漫画字体，字体内容“元气满满”，漫画纹理，胖嘟嘟可爱感觉，黑色背景。卡通感十足。
